# History of Git

Git is a version control system widely used for tracking changes in source code.

## 3 Facts About Git

1. **Created by Linus Torvalds**  
   Git was created in 2005 by Linus Torvalds, the creator of Linux, to help manage Linux kernel development.

2. **Distributed Version Control**  
   Git allows multiple developers to work on a project simultaneously without needing a central server.

3. **Supports Branching and Merging**  
   Git makes it easy to create branches for new features and merge them back into the main codebase.
